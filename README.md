# 硬件实训-智能小车
### 红外遥控按键说明 (InfraredDriver.c)
- 电源键：启动避障程序
- 灯光键：启动循迹程序
- 方向键：直行、后退、原地转向（可调距离）
- 左转右转键：旋转90°
- +、-键：左转、右转（可调距离）
- 声音按钮：喇叭
- 0：直行0.5s
- 1、2、3：调节可调的移动时长，分别为0.05s、0.1s、0.5s
- 8：直行3.5s
- 9：中止

遥控改键请前往`Driver/InfraredDriver.c`和`User/global.h`

### 避障功能说明 (Avoid.c)
使用前先校准传感器的触发阈值。
逻辑为小车贴着左侧墙走，⚠️需要手动调整红外避障传感器指向，左侧的朝向正左方，**右侧的朝向正前方**。屏幕显示`F:1`指**右传感器**即前方有障碍，显示`L:1`指**左传感器**即左侧有障碍。

### 循迹功能说明 (Track.c)
使用前先校准四路循迹传感器的触发阈值。
循迹参数调整前往`Driver/Track.c`
准备了**慢速版、快速版、单路传感器版**三种方案，可在Track_Start()中修改。
